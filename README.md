## Flapy Bird Clone
This game is a simple flappy bird clone created as a learning process.

## Technologies Used
* JavaScript 
* Apache Cordova
* Phaser

## Install
Clone the repo, open in Visual Studio 2015 (Ensure Tools for Apache Cordova are installed) and hit Run!
